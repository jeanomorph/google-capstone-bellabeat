**Bellabeat Case Study (Google Data Analytics Capstone)**

**Author:** Jeanelfe Joy Delfinado

**Output:** HTML (.html)

**Tools Used:** R, tidyverse, lubridate, janitor, ggplot2

**Overview**

This project analyzes Fitbit user activity and sleep data to generate insights for Bellabeat.
The goal is to understand user habits and support marketing strategy for the Leaf wellness tracker.

The HTML report includes:

* Data cleaning
* Transformation
* Visualizations
* Insights
* Recommendations
* Full code (echo = TRUE)

**Packages**

* tidyverse for data wrangling
* lubridate for date handling
* janitor for cleaning
* ggplot2 for visualization
